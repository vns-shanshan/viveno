# Viveno

## Introduction:

Viveno is a web application that helps users discover events and attend them easily.
Users can browse events, explore details, and engage with activities that match their interests.

## Command to run the app:

#### backend:

`cd backend`
`npm run dev`

#### frontend:

`cd frontend`
`npm run dev`
